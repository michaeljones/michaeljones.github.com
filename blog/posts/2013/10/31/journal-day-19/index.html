  • Written a basic example of working with the IORef type:

    import qualified Data.IORef as R

main = do

    -- Create a new IORef for handling a mutable number
    intioref <- R.newIORef 5

    -- Read the number
    i <- R.readIORef intioref

    -- Print the number
    print i

    -- Increment the number in place
    R.writeIORef intioref $ succ i

    -- Prin the number
    R.readIORef intioref >>= print

    return ()

    I need to dig some more to attempt to understand what is actually going on and the best way to think about it.
